# Community Uniswap SDK

This community SDK for Uniswap implements support for the following features:

- Uniswap V4
- Permit2

## Contributors & Maintainers

- Zaha Studio
  - @akshatmittal
- BootNode
  - @luchobonatti
  - @nicosampler

## Installation

```bash
pnpm install @zahastudio/uniswap-sdk viem
```

If you need access to native `@uniswap/*` packages, you can install them as well. The package enforces minimum versions to ensure compatibility.
